# 量刑制度

## 累犯
- **一般累犯**：故意犯罪 → 有期徒刑以上 → 5年内再犯故意罪 → 有期徒刑以上
- **特殊累犯**：危害国家安全/恐怖活动/黑社会性质犯罪，任何时候再犯
- 从重处罚，不得缓刑、不得假释

## 自首
- **一般自首**：自动投案 + 如实供述
- **特别自首**：被采取强制措施后，如实供述司法机关未掌握的**其他罪**
- 可以从轻或减轻；犯罪较轻的，可以免除

## 立功
- **一般立功**：揭发他人犯罪、提供重要线索等
- **重大立功**：揭发重大犯罪、有重大发明等
- 立功可以从轻或减轻；重大立功可以减轻或免除

## 数罪并罚
- 管制最高3年，拘役最高1年，有期徒刑最高20年（总和不满35年）或25年（总和35年以上）
- 附加刑仍需执行

## 缓刑
- 条件：拘役或3年以下有期徒刑 + 犯罪情节轻 + 悔罪 + 无再犯危险
- 缓刑考验期：拘役为原刑期+1年（≥2个月），有期徒刑为原刑期+5年（≥1年）
- 不包括累犯和犯罪集团首要分子
