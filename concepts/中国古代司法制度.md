# 中国古代司法制度

## 各朝代特色制度

| 朝代 | 制度 | 要点 |
|:----|:----|:----|
| 西周 | 狱讼分理 | 刑/民分开，三刺制度 |
| 秦 | 爰书 | 案件调查笔录 |
| 唐 | **三司推事** | 大理寺+刑部+御史台会审 |
| 宋 | 审刑院 | 皇帝直接控制司法 |
| 明 | 三司会审、廷杖、厂卫 | 厂卫特务干预司法 |
| 清 | **秋审**（最重要！） | 情实/缓决/可矜/可疑/留养承祀 |
| 根据地 | 马锡五审判方式 | 深入群众、调解为主 |

## 秋审五类结果
> **"情实缓决矜可疑，留养承祀也记一"**
1. 情实 → 执行
2. 缓决 → 减等
3. 可矜 → 减等
4. 可疑 → 减等
5. 留养承祀 → 免死

## 监察制度沿革
御史台（唐）→ 都察院（明清）→ 监察委（现代）
