# 清末修律（沈家本修律）

## 背景
晚清内忧外患，领事裁判权（列强借口中国法律落后不交还）倒逼法制改革。

## 领军人物
**沈家本** — 清末修律主持者，主张"参考古今，博稽中外"

## 主要成果
| 成果 | 意义 |
|:----|:----|
| 《大清现行刑律》 | 过渡性法典，删除酷刑 |
| **《大清新刑律》** | **第一部近代化刑法典**，吸收西方刑法原则 |
| 《大清民律草案》 | 第一部民法草案（未实施） |
| 《大清商律草案》 | 商事立法尝试 |
| 废除凌迟/枭首/戮尸 | 刑罚人道化 |
| 改良监狱 | 近代监狱制度开端 |

## 宪法性文件
| 文件 | 特点 |
|:----|:----|
| 《钦定宪法大纲》（1908） | 君上大权+臣民权利义务 |
| 《十九信条》（1911） | 武昌起义后紧急立宪，限制皇权 |
