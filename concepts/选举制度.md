# 选举制度

## 基本原则
1. **普遍性：** 年满18岁中国公民，除依法剥夺政治权利外都有选举权
2. **平等性：** 一人一票，同票同权
3. **直接选举与间接选举并用：**
   - 县级/乡级人大代表 → 直接选举
   - 全国/省级/市级人大代表 → 间接选举
4. **秘密投票：** 无记名投票

## 重点数据
| 项目 | 数值 |
|:----|:----|
| 选举法定年龄 | 18岁 |
| 直接选举选区划分 | 每一代表所代表人口数大致相等 |
| 代表当选条件 | 全体选民过半数参加投票 + 获得参加投票选民过半数票 |
