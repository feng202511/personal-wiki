# 历代主要法典

## 里程碑法典

| 朝代 | 法典 | 意义 |
|:----|:----|:----|
| 战国 | 《法经》（李悝） | 中国第一部成文法典 |
| 秦 | 秦律（商鞅改法为律） | 第一部"律"形式 |
| 汉 | 《九章律》（萧何） | 汉律体系基础 |
| 三国 | 《魏律》（陈群） | 八议入律，体例创新 |
| 北齐 | 《北齐律》 | 12篇体例，名例律为首 |
| **唐** | **《唐律疏议》** | 现存最早最完整的封建法典，一准乎礼 |
| 宋 | 《宋刑统》 | 第一部刊印颁行的法典 |
| 明 | 《大明律》 | 7篇体例，简于唐律 |
| 清 | 《大清律例》 | 最后一部封建法典 |
| 清末 | 《大清新刑律》 | 第一部近代化刑法典 |

## 记忆口诀
> **"法经秦律汉九章，魏律北齐开新篇；唐律宋统明七篇，大清律例终封建"**
