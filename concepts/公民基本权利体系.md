# 公民基本权利体系

## 七大权利板块

| 类型 | 具体权利 | 高频考点 |
|:----|:--------|:--------|
| ⚖️ 平等权 | 法律面前平等 | 合理差别 vs 歧视 |
| 🗳️ 政治权利 | 选举权/被选举权、言论出版集会结社游行示威 | 谁没有选举权 |
| 🙏 宗教信仰自由 | 信仰自由、宗教活动自由 | 宗教与教育分离 |
| 🧍 人身自由 | 人身不受侵犯、人格尊严、住宅、通信 | 逮捕条件、搜查程序 |
| 💰 社会经济权利 | 财产权、劳动权、休息权、受教育权、获得物质帮助权 | 权利义务双重性 |
| 🎭 文化权利 | 科研创作自由、文化生活参与权 |  |
| 👁️ 监督权 | 批评建议、申诉控告检阅 | 与国家赔偿关系 |

## 记忆口诀
> **"平政宗人财文监"** — 七大权利类别
